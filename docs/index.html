MediaLab – Extracción y Procesamiento de Datos

Este proyecto demuestra un flujo básico de:

  • Web scraping de titulares (Python Blog)
  • Procesamiento de texto
  • Cálculo de frecuencias de palabras
  • Publicación de datos en GitHub Pages

  • Descargar: top_palabras_titulos.csv
